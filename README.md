## Installation Guide
### 1. Add Repo
```bash
helm repo add datafyio https://helm.datafy.io/datafy-agent
helm repo update
```

### 2. Install
```bash
helm install datafy-agent --version 3.2.3 datafyio/datafy-agent \
--namespace datafy-agent --create-namespace \
--set agent.mode="sensor/autoscaler" \
--set agent.token=<your_token> \
--atomic
```

## Upgrade
```bash
helm repo update
helm upgrade --install datafy-agent --version 3.2.3 datafyio/datafy-agent -n <namespace> --reuse-values --atomic
```

Switch mode:
```bash
helm upgrade --install datafy-agent --version 3.2.3 datafyio/datafy-agent -n <namespace> --set agent.mode=autoscaler --atomic
```

Rollback:
```bash
helm rollback datafy-agent 1 -n <namespace>
```

### 5. Verify
```bash
kubectl get pods -n <namespace> -l app.kubernetes.io/instance=datafy-agent
```
